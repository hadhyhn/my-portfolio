Gait Recognition and Classification for Surveillance

  • This project aims to develop a comprehensive gait recognition model using deep learning techniques that fuses multiple modalities of gait data, leveraging both Convolutional Neural Networks (CNN) and Graph Convolutional Networks (GCN). By combining these techniques, the model can effectively capture spatial and temporal features from various gait sources, enabling robust and accurate identification and analysis of individuals based on their unique walking patterns.

Anti theft Flooring System

  • The secure flooring tile system is an IoT-based home security solution powered by Arduino. When activated, the system monitors the house using two secure flooring tiles equipped with Piezo sensors. When someone unauthorized enters the house and steps on the floor, the Piezo sensor detects the pressure and triggers the Arduino to send an immediate message over the IoT network. This message notifies the homeowner or relevant authorities, enabling a quick response to potential theft or intrusion, enhancing the security of the premises.

Traffic Sign Recognition

  • Developed a traffic sign recognition system using a CNN model. Facial Recognition Door Lock

  • Worked on a facial recognition door lock system using Raspberry Pi. To enable cloud communication, I used AWS IoT Core and implemented HTTP-based publish/subscribe messaging. When a recognized face was detected, AWS IoT sent an HTTP message to the Raspberry Pi to unlock the door, providing a seamless and secure access control solution.
